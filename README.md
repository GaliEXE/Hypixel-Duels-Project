# Netlify Identity Demo in Vuejs

## Instroduction

Features:
 - Vue.js 2.x
 - vuex
 - vue-router
 - Netlify Identiy Widget
 - Netlify (hosting) 

Upgrade:
 - Vue.js 3.x in 2019
 - Keep upgrading all libraries when neccessary

## Project setup
```
yarn install
```

### For development
```
yarn serve
```

### For production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
